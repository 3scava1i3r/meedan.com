    General

    variables

    width--primary-logo

    $width--primary-logo: 200px;

    Description

    Branding

    At the top of homepage and product pages, this usually has an inverted color scheme, a large .logo image and the h1. At smaller sizes the logo has a percent-based width.

    logo-small-width

    $logo-small-width: 150px;

    Description

    Tile

    A flex-based layout that handles many small blocks. It is implemented in several places. There is custom width handling on each image, where needed.

    Appears on several pages.

    Example

    <div class="logos">
  <span><img src="http:placehold.it/80x80" /></span>
  <span>2</span>
  <span>3</span>
</div>

    breakpoint

    $breakpoint: 900px;

    Description

    Responsive configuration

    These are global styles that can be used to theme the app. Currently the setup is very simple. The breakpoint definitions are used in media queries. like @media all and (max-width: $breakpoint) { .. styles for small screens .. }

    item-width

    $item-width: 160px;

    Description

    Homepage team grid component

    We use breakpoints to control how many items wide it is.

    This is another flex-based layout, similar to the logos component.
